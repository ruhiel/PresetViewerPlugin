# PresetViewerPlugin
プリセット一覧を表示一覧を表示する KanColleViewer 用のプラグインです。

### インストール

* `PresetViewerPlugin.dll` を KanColleViewer の `Plugins` ディレクトリに放り込んで下さい。

### アンインストール

* KanColleViewer が起動していない状態で`PresetViewerPlugin.dll`と KanColleViewer がインストールされている
  ディレクトリの `PresetViewPlugin.txt` を 削除して下さい。
